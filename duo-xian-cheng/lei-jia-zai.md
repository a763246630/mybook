# 类加载机制





### 类加载过程

类加载到使用整个过程有如下几步： 

**加载 >> 验证 >> 准备 >> 解析 >> 初始化 >>** 使用 >> 卸载	

#### 加载

在硬盘上查找并通过IO读入字节码文件，使用到类时才会加载，例如调用 

类的main()方法，new对象等等 

#### 验证

校验字节码文件的正确性 

#### 准备

给类的静态变量分配内存，并赋予默认值 

#### 解析

将符号引用替换为直接引用，该阶段会把一些静态方法(符号引用，比如 

main()方法)替换为指向数据所存内存的指针或句柄等(直接引用)，这是所谓的**静态链** 

**接**过程(类加载期间完成)，**动态链接**是在程序运行期间完成的将符号引用替换为直接 

引用，下节课会讲到动态链接 

#### 初始化

对类的静态变量初始化为指定的值，执行静态代码块
